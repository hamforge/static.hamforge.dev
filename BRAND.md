# BRAND.md

## Source of truth
This brand system follows the current hamforge and Josh Benham landing pages.

## Core direction
- flat UI
- dark slate foundation
- orange primary accent
- white headings
- muted slate body text
- rounded cards and panels
- subtle hover polish
- clean Tailwind utility styling
- no gradients unless explicitly requested
- lighter strokes/borders on boxes

- Inter
- fallback: Arial, Helvetica, sans-serif

## hamforge rules
- single-page landing style
- logo left, email top-right
- product/studio preview on the right
- keep hero/product panel believable and app-like
- preserve current copy and layout unless explicitly changed

## Josh Benham rules
- simpler personal/founder portfolio
- preserve current copy and layout unless explicitly changed
- same visual system as hamforge, but more understated

## Buttons
Primary:
- orange fill
- dark text
- rounded-xl
- subtle lift on hover

Secondary:
- dark/slate border
- white/slate text
- subtle hover brighten

## Cards
- rounded-2xl or rounded-3xl
- lighter slate borders
- dark panel fill
- flat design
- subtle hover lift only

## AI / Codex edit rules
1. Preserve Tailwind utility class style.
2. Preserve the dark slate + orange palette.
3. Do not introduce gradients, glossy effects, or decorative clutter.
4. Keep interactions subtle.
5. Prefer hierarchy and spacing improvements over adding more elements.
6. Keep logo areas easy to swap with real assets later.
